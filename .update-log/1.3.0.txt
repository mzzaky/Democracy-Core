    CUSTOM GUI ENHANCEMENTS

2. CUSTOM GUI UPDATE
   - Menambahkan dokumentasi (comment help) pada bagian header gui.yml
   - Implementasi fungsi click_sound yang sudah dikonfigurasi di gui.yml
      - Setiap item di GUI kini memiliki audio feedback saat diklik
      - Dynamic sound berdasarkan state (active/inactive, ada/kosong)
   - Menambahkan parameter gui_slot ke semua item di gui.yml
      - Posisi slot sekarang dapat dikustomisasi tanpa mengubah kode Java
   - Sistem baru untuk menyembunyikan item flags/attributes pada GUI items
      - Single value support: hide_attributes: ALL
      - List support: hide_attributes:
          - ENCHANTS
          - ATTRIBUTES
      - All ItemFlags supported (HIDE_ENCHANTS, HIDE_ATTRIBUTES, dll)
      - State-aware configuration (bisa beda untuk active/inactive state)
    - PLACEHOLDERAPI SUPPORT IN GUI
      - Full support for PlaceholderAPI placeholders in gui.yml
      - Can be used in display_name, lore, and command values
      - Initializes automatically if PlaceholderAPI is present
      - Supports both internal placeholders ({player_name}) and PAPI placeholders (%democracy_president%)
   - CUSTOMIZABLE GUI SIZE
      - Menambahkan opsi inventory_slot pada gui.yml
      - Administrator kini dapat mengubah ukuran Main Menu GUI (9, 18, 27, 36, 45, 54 slot)
      - Sistem auto-adapt untuk dekorasi corner sesuai ukuran GUI
      - Validasi error check untuk memastikan ukuran GUI valid (kelipatan 9)
   - Sistem GUI yang lebih fleksibel dengan konfigurasi on_click di gui.yml
      - Setiap item kini dapat dikustomisasi untuk membuka GUI berbeda atau menjalankan special actions.
      - Administrator dapat mengubah fungsi item tanpa perlu edit code Java
      - Mendukung multiple states (default/no_president, active/inactive)

      Available Actions and Special Action:
      
      === OPEN GUI ACTIONS (11) ===
      - <open_gui_main_menu>          : Opens Main Menu GUI
      - <open_gui_player_stats>       : Opens Player Statistics GUI
      - <open_gui_government>         : Opens Government GUI
      - <open_gui_voting>             : Opens Voting/Election GUI
      - <open_gui_cabinet>            : Opens Cabinet GUI
      - <open_gui_treasury>           : Opens Treasury GUI
      - <open_gui_executive_orders>   : Opens Executive Orders GUI
      - <open_gui_recall>             : Opens Recall System GUI
      - <open_gui_history>            : Opens Presidential History GUI
      - <open_gui_leaderboard>        : Opens Leaderboard GUI
      - <open_gui_help>               : Opens Help/Guide GUI
      
      === SPECIAL ACTIONS (7) ===
      - <action_register_candidate>   : Register as presidential candidate
      - <action_rate_president>       : Show rating command message
      - <action_close_inventory>      : Close inventory
      - <action_arena_info>           : Show arena information
      - <action_endorse_candidate>    : Show endorse command message
      - <action_donate_treasury>      : Show donate command message
      - <action_join_arena>           : Join presidential arena
      - <action_refresh_gui>          : Refresh GUI
      - <action_console_command>      : Run console command
