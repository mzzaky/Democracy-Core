Version 1.3.0

Custom GUI & Placeholders
February 16, 2026
▼ Minimize
Release Overview

This update introduces comprehensive PlaceholderAPI support with over 40 new placeholders, a fully customizable GUI system with dynamic sound effects, adjustable sizes, and flexible item configurations, alongside significant refactoring and bug fixes.

Features Added
15+
System
GUI / API
Type
Update
+
Added - PlaceholderAPI Integration
  • PlaceholderAPI Support
    Added soft dependency for PlaceholderAPI integration.
  • Comprehensive Placeholders
    Implemented 40+ placeholders covering various game aspects:
    Government: %democracy_president%, %democracy_approval_rating%, etc.
    Election: %democracy_phase%, %democracy_total_votes%, etc.
    Treasury: %democracy_treasury%, %democracy_treasury_income%, etc.
    Player Stats: %democracy_votes_cast%, %democracy_playtime_hours%, etc.
    Arena: %democracy_arena_kd%, %democracy_killstreak%, etc.

    TAB Plugin & HolographicDisplays
    DeluxeMenus & Essentials/CMI

Improved - Custom GUI System
  • Click Sound Feedback
    Implemented configurable audio feedback when clicking GUI items.
  • Customizable GUI Size
    Admins can now configure the Main Menu size (9, 18, 27, 36, 45, or 54 slots).
    Added auto-adaptation for corner decorations based on size.
    Included smart validation to ensure valid slot counts.
  • Flexible Item Slotting
    Added gui_slot parameter to move items without Java code changes.
  • Hide Attributes Control
    New system to hide ItemFlags (Attributes, Enchants, etc.) via config.
  • Placeholder Support in GUI
    Full PAPI support in Display Names, Lore, and Commands within the GUI.
  • Advanced Action System
    New on_click configuration allows defining actions like:
    <open_gui_...> to open specific menus (Government, Treasury, etc.)
    <action_register_candidate>, <action_donate_treasury>, etc.
    <action_console_command> for executing server commands.
  • Configuration Documentation
    Added comprehensive comment documentation to the gui.yml header.
⚡
Refactored - Quick Actions
  • Configurable Quick Actions
    Register, Vote, and Rate items are now fully configurable in gui.yml.
  • Dynamic Requirements
    Added placeholders for requirements in GUIs:
    {req_level}, {req_playtime}, {req_balance}, {cost}
🔧
Fixed - Bug Fixes
  • GUI Decoration Logic
    Fixed hardcoded glass pane materials; now correctly reads from config.
    Added support for AIR material to remove decorations.
  • Active Effects Item
    Fixed issues with hardcoded materials and attributes for active status items.

Removed - Legacy Features
  • Quick Actions Menu
    Removed the separate Quick Actions GUI as it is now redundant with the Main Menu improvements.

Version 1.0.0

Initial Release
December 19, 2025
▼ Minimize
Release Overview

Democracy Core is a comprehensive democracy and government simulation plugin for Minecraft servers. This release introduces a full-featured democratic system with elections, government management, executive powers, cabinet decisions, and community events.

Features Added
30+
Systems
5+
Type
Major
+
Added - Core Government System
  • Presidential Elections: Multi-phase election system (Registration → Campaign → Voting → Inauguration)
  • Cabinet System: 5 ministries (Defense, Treasury, Commerce, Infrastructure, Culture) with appointed ministers
  • Approval Ratings: Players can rate the president (1-5 stars) affecting government performance
  • Term Limits: Presidents serve 30-day terms with automatic elections
  • Government History: Track all past presidents and their achievements
+
Added - Election Mechanics
  • Candidate Registration: Level/playtime/balance requirements with campaign deposits
  • Endorsement System: Candidates gain support through player endorsements
  • Weighted Voting: Vote weight based on player level, playtime, and balance
  • Campaign Broadcasting: Candidates can broadcast campaign messages during campaign phase
  • Voter Rewards: Participation rewards and lottery system for voters
  • Automatic Elections: Elections trigger automatically before term end or when no president exists
+
Added - Executive Powers
  • Executive Orders: President can issue powerful orders with treasury costs and cooldowns:
    Golden Age: XP/Vault multipliers, rare drop bonuses
    State of Emergency: Disables PvP server-wide
    National Holiday: Clears player cooldowns
    War Economy: Doubles PvP rewards
    Economic Recovery: Stimulus payments and shop discounts
    Infrastructure Initiative: Haste effects for all players
    Environmental Protection: Farming multipliers
    Education Advancement: XP multipliers
    Purge Protocol: Full PvP everywhere
    Presidential Pardon: Clears player punishments
+
Added - Cabinet Decisions
  • Defense Ministry
    War Games, Military Draft, National Protection, Armory Discount, Border Patrol
  • Treasury Ministry
    Tax Holiday, Economic Stimulus, Auction Boost, Treasury Bonus, Market Crash
  • Commerce Ministry
    Trade Fair, Resource Boom, Merchant Caravan, Black Friday, Treasure Hunt
  • Infrastructure Ministry
    Construction Boom, Free Real Estate, Builder's Paradise, Public Works, Infrastructure Grants
  • Culture Ministry
    Festival Week, Double XP Weekend, Firework Celebration, Cultural Exchange, Community Event
+
Added - Arena & Recall System
  • Presidential Arena: 7-day PvP events, killstreaks, leaderboards, grand prizes
  • Recall System: Petition creation, signature collection (30%), recall voting (60% to remove)
!
Technical & Player Features
  • Treasury Management: Audit trails, automatic salaries, donations
  • Player Features: Playtime tracking, level calculation, daily rewards
  • Admin Tools: Force elections, treasury control, data management
  • Dependencies: Vault (for economy)
➜
Future Plans & Roadmap
  • International relations system
  • Political parties
  • Advanced diplomacy features
  • Custom election maps
  • Mobile voting support
