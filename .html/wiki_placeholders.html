Overview

Integration

DemocracyCore provides complete integration with **PlaceholderAPI**, allowing you to display government data, election status, player statistics, treasury balance, and arena info in other plugins like scoreboards, chat, tab lists, and holograms.

Requirements
  • PlaceholderAPI v2.11.6+
  • DemocracyCore v1.3.0+

Installation

/papi ecloud download PlaceholderAPI
/papi reload
# Test
/papi parse me %democracy_president%

Government Placeholders

Global

President & Term Info

Placeholder Description Return Type
%democracy_president% Current president's name String
%democracy_term_time% Remaining term time Formatted String
%democracy_approval_rating% Approval rating (1.0 - 5.0) Decimal
%democracy_consecutive_terms% Number of consecutive terms Integer
%democracy_salary_payouts% Total salary paid this term Decimal

Cabinet Ministers

Placeholder Description
%democracy_defense_minister% Minister of Defense name
%democracy_treasury_minister% Minister of Treasury name
%democracy_commerce_minister% Minister of Commerce name
%democracy_infrastructure_minister% Minister of Infrastructure name
%democracy_culture_minister% Minister of Culture name

Usage Examples

Scoreboard (FeatherBoard)

lines:
  - "&ePresident: &f%democracy_president%"
  - "&eRating: &f%democracy_approval_rating%"
  - "&aElection: &f%democracy_phase%"
  - "&aTime: &f%democracy_phase_time%"
  - "&bRole: &f%democracy_cabinet_position%"

Chat Prefix (Vault/LP)

# Conditional Formatting
prefix: "%democracy_is_president:&6[President] :%%democracy_cabinet_position%"

Troubleshooting

Placeholders not parsing?
  1. Ensure DemocracyCore is enabled in /plugins.
  2. Run /papi list and check if democracy is listed.
  3. Try /papi reload.
  4. Verify you are using the correct placeholder naming.
